생태계

OpenCode로 구축된 프로젝트와 통합.

OpenCode를 기반으로 만들어진 커뮤니티 프로젝트 모음입니다.

참고

이 목록에 OpenCode 관련 프로젝트를 추가하고 싶다면 PR을 제출하세요.

awesome-opencode와 opencode.cafe도 함께 확인해 보세요. ecosystem과 community 정보를 한곳에 모아볼 수 있습니다.

플러그인

이름설명
opencode-daytonagit sync와 live preview를 지원하는 격리된 Daytona sandbox에서 OpenCode 세션을 자동 실행합니다.
opencode-helicone-session요청을 그룹화할 수 있도록 Helicone session header를 자동으로 주입합니다.
opencode-type-inject조회 tool과 함께 TypeScript/Svelte 타입 정보를 파일 읽기에 자동 주입합니다.
opencode-openai-codex-authAPI 크레딧 대신 ChatGPT Plus/Pro 구독을 사용할 수 있습니다.
opencode-gemini-authAPI 과금 대신 기존 Gemini 플랜을 사용할 수 있습니다.
opencode-antigravity-authAPI 과금 대신 Antigravity의 무료 model을 사용할 수 있습니다.
opencode-devcontainersshallow clone과 자동 포트 할당을 기반으로 multi-branch devcontainer 격리를 제공합니다.
opencode-google-antigravity-authGoogle Search 지원과 견고한 API 처리를 제공하는 Google Antigravity OAuth Plugin입니다.
opencode-dynamic-context-pruning오래된 tool output을 정리해 token 사용량을 최적화합니다.
opencode-vibeguardLLM 호출 전에 secrets/PII를 VibeGuard 스타일 placeholder로 가리고, 로컬에서 복원합니다.
opencode-websearch-cited지원 provider에서 Google grounded 스타일의 네이티브 websearch를 추가합니다.
opencode-ptyAI agent가 PTY에서 백그라운드 프로세스를 실행하고 대화형 입력을 보낼 수 있게 합니다.
opencode-shell-strategy비대화형 shell 명령 실행 지침을 제공해 TTY 의존 작업으로 인한 멈춤을 방지합니다.
opencode-wakatimeWakatime으로 OpenCode 사용량을 추적합니다.
opencode-md-table-formatterLLM이 생성한 markdown 표를 정리합니다.
opencode-morph-pluginMorph를 통한 Fast Apply 편집, WarpGrep 코드베이스 검색 및 컨텍스트 압축
oh-my-opencodebackground agent, 사전 구성된 LSP/AST/MCP tool, curated agent, Claude Code 호환성을 제공합니다.
opencode-notificatorOpenCode 세션에 데스크톱 알림과 사운드 알림을 제공합니다.
opencode-notifierpermission, 완료, 오류 이벤트에 대한 데스크톱 알림과 사운드 알림을 제공합니다.
opencode-zellij-namerOpenCode 맥락을 기반으로 Zellij session 이름을 AI로 자동 지정합니다.
opencode-skillfulskill 탐색과 주입을 통해 OpenCode agent가 필요 시 prompt를 lazy load하도록 합니다.
opencode-supermemorySupermemory를 사용해 세션 간 persistent memory를 제공합니다.
@plannotator/opencode시각 주석과 private/offline 공유를 포함한 인터랙티브 계획 리뷰를 제공합니다.
@openspoon/subtask2세밀한 flow control로 opencode /commands를 강력한 orchestration 시스템으로 확장합니다.
opencode-schedulercron 문법을 사용해 launchd(Mac) 또는 systemd(Linux) 기반 반복 작업을 예약합니다.
micodeStructured Brainstorm → Plan → Implement 워크플로를 session continuity와 함께 제공합니다.
octto다중 질문 폼 기반의 AI 브레인스토밍용 인터랙티브 브라우저 UI를 제공합니다.
opencode-background-agentsClaude Code 스타일의 background agent를 async delegation과 context persistence로 제공합니다.
opencode-notifyOpenCode 작업 완료 시점을 native OS 알림으로 알려줍니다.
opencode-workspace16개 구성요소를 한 번에 설치하는 bundled multi-agent orchestration harness를 제공합니다.
opencode-worktreeOpenCode용 git worktree를 손쉽게 사용할 수 있도록 돕습니다.
opencode-sentry-monitorSentry AI 모니터링으로 AI 에이전트를 추적하고 디버그합니다.
