# [First steps with uv](#first-steps-with-uv)

After [installing uv](https://jiezhi.github.io/dochub/uv/getting-started/installation/index.md), you can check that uv is available by running the `uv` command:

```
$ uv
An extremely fast Python package manager.

Usage: uv [OPTIONS] <COMMAND>

...
```

You should see a help menu listing the available commands.

## [Next steps](#next-steps)

Now that you've confirmed uv is installed, check out an [overview of features](https://jiezhi.github.io/dochub/uv/getting-started/features/index.md), learn how to [get help](https://jiezhi.github.io/dochub/uv/getting-started/help/index.md) if you run into any problems, or jump to the [guides](https://jiezhi.github.io/dochub/uv/guides/index.md) to start using uv.
